# Portfolio · Olatz González

Portfolio web de una sola página construido con React + Vite.

UX/UI Designer y Frontend Developer en Bilbao.

## Stack

- React 18 + Vite
- CSS Modules (sin librerías UI externas)
- Accesibilidad WCAG 2.1 AA

## Secciones

- **Hero** — presentación y redes sociales
- **Proyectos** — SignaTour, IsWorking, SustraiApp, Dragones y Mazorcas
- **Sobre mí** — perfil y datos destacados
- **Skills** — diseño, frontend, backend y herramientas
- **Contacto** — email, LinkedIn y descarga de CV

## Desarrollo local

```bash
npm install
npm run dev
```
